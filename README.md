# Budget Tracker CLI

Интерактивный CLI-контроллер для управления личными финансами с удобным цифровым меню.

## 📋 О проекте

Приложение позволяет управлять личными финансами через командную строку. Создавайте счета, добавляйте транзакции, отслеживайте баланс и экспортируйте данные в CSV.

## ✨ Возможности

- 📊 **Просмотр всех счетов** с балансами
- ➕ **Создание новых счетов**
- 👁️ **Детальный просмотр счета** со списком транзакций
- 💰 **Добавление транзакций** (доходы и расходы)
- 🗑️ **Удаление транзакций**
- 📤 **Экспорт транзакций в CSV**
- ❌ **Удаление счетов**

## 🚀 Установка и запуск

### Предварительные требования
- [Node.js](https://nodejs.org/) (версия 14 или выше)
- npm (устанавливается вместе с Node.js)

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/budget-tracker-cli.git

# Перейти в папку проекта
cd budget-tracker-cli

# Установить зависимости
npm install
